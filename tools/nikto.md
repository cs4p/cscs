# nikto
Scan web server for known vulnerabilities 

https://github.com/sullo/nikto

# Installation
## MacOS
    brew install nikto
## Debian
    apt install nikto

# Examples

# Basic scan
	nikto -host 192.168.1.1 -port 443
# Update databases and plugins from CIRT.net
	nikto -update
